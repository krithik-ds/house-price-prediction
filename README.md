# House Price Prediction using Machine Learning

##  Overview

This project predicts house prices using a Random Forest Regression model based on features such as area, number of rooms, and year built.

## Dataset

* Source: Kaggle Housing Prices dataset

## Model Used

* Random Forest Regressor

## Results

* Validation MAE: 21,857

## Visualization

Includes Actual vs Predicted price scatter plot to evaluate model performance.

## Files

* `house_price_prediction.ipynb` → Main notebook
* `submission.csv` → Final predictions

## Conclusion

The model performs reasonably well and can be improved with feature engineering and hyperparameter tuning.
